# 医院运营决策分析系统（DRM）详细产品需求文档

> **版本**：v4.0
> **日期**：2026-05-22
> **产品定位**：融合主流厂商优势 + AI大模型增强 + 语音交互的院长运营决策平台
> **核心理念**：取各家之长 + AI差异化创新 + 自然语言交互

---

## 一、产品定位与竞争策略

### 1.1 主流厂商优势分析

| 厂商 | 核心优势 | 本产品吸收要点 |
|------|----------|----------------|
| **用友医疗** | 财务一体化、集团化管控、预算管理 | 业财融合、多院区管理 |
| **望海康信** | 成本核算精细化、绩效管理专业 | 四级分摊法、RBRVS/DRG绩效 |
| **金蝶医疗** | 云原生、移动化、AI智能 | 移动院长驾驶舱、智能问答 |
| **卫宁健康** | HIS深度集成、临床融合 | 数据实时同步、临床运营联动 |
| **创业软件** | 公立医院绩效考核、DRG适配 | 国考指标体系、DRG盈亏分析 |
| **熙软科技** | 业财融合、智能分析 | 实时数据、智能预警 |
| **东软集团** | 大数据平台、区域医疗 | 数据仓库、医联体管理 |

### 1.2 本产品核心定位

**传统厂商能做的，我们做得更好；传统厂商做不到的，我们用AI突破**

| 维度 | 传统厂商 | 本产品 |
|------|----------|--------|
| **数据整合** | HIS/财务/手术等系统对接 | 同卫宁，支持实时同步 |
| **财务管控** | 用友强项 | 业财融合，借鉴用友 |
| **成本核算** | 望海最强 | 四级分摊法，借鉴望海 |
| **绩效管理** | 望海/创业软件强 | RBRVS+DRG双模式 |
| **移动化** | 金蝶领先 | 移动院长驾驶舱，借鉴金蝶 |
| **大屏展示** | 各厂商都有 | 院长视角，一键看懂 |
| **语音交互** | **无** | **行业首创，语音查询+控制** |
| **AI智能分析** | 传统厂商弱 | AI归因分析，差异化亮点 |
| **知识沉淀** | 传统厂商无 | 知识库RAG，差异化亮点 |
| **智能交付** | 传统厂商无 | 自动生成报告，差异化亮点 |

## 三、详细菜单功能设计

### 3.1 目录一：院长驾驶舱（可视化大屏）

**定位**：院长日常查看核心指标的第一入口，一屏掌握全院运营状况

**设计理念**：借鉴金蝶移动驾驶舱 + 望海数据可视化 + 创业软件国考指标

#### 3.1.1 院长驾驶舱首页

**功能描述**：
院长登录系统后的默认首页，以大屏形式展示医院核心运营指标。支持多屏投放、全屏展示、语音唤醒。

**核心功能**：

| 功能点 | 说明 |
|--------|------|
| 核心指标卡片 | 门诊人次、住院人次、手术台次、医疗收入四大核心指标实时展示 |
| 同比环比分析 | 与昨日、上周、上月、去年同期对比，以箭头和百分比展示 |
| 床位效率监控 | 全院床位使用率、平均住院日、周转次数，紧张科室TOP3预警 |
| 医疗质量概览 | 病历质控率、处方合格率、不良事件、低风险死亡率 |
| DRG核心指标 | CMI值、RW值、入组率、结余率 |
| 收入结构分析 | 门诊收入、住院收入占比，科室收入排名 |
| 成本概览 | 本月成本、成本率、成本结构占比 |
| 预警信息 | 实时预警信息展示，支持红色/黄色/蓝色分级 |
| 语音入口 | 右上角语音按钮，支持语音查询和控制 |

**数据要求**：

| 数据项 | 数据来源 | 刷新频率 |
|--------|----------|----------|
| 门诊人次 | HIS系统 | 5分钟 |
| 住院人次 | HIS系统 | 5分钟 |
| 手术台次 | 手术系统 | 5分钟 |
| 医疗收入 | 财务系统 | 5分钟 |
| 床位数据 | HIS系统 | 5分钟 |
| 质量数据 | 质控系统 | 30分钟 |
| DRG数据 | DRG系统 | 日更 |

#### 3.1.2 运营趋势分析

**功能描述**：
展示医院运营指标的时间趋势，支持日/周/月/年多时间维度切换，多指标叠加对比。

**核心功能**：

| 功能点 | 说明 |
|--------|------|
| 多指标曲线 | 门诊人次、住院人次、手术台次、医疗收入曲线叠加展示 |
| 时间维度切换 | 日视图（按小时）、周视图（按天）、月视图（按天）、年视图（按月） |
| 同比对比 | 与历史同期数据叠加对比，虚线表示历史，实线表示当前 |
| 指标筛选 | 可选择显示/隐藏某些指标曲线 |
| 缩放平移 | 支持鼠标滚轮缩放、拖拽平移 |
| 数据标注 | 鼠标悬停显示具体数值，支持标注特殊事件 |
| 语音控制 | "切换到月视图"、"显示收入趋势"等语音指令 |

#### 3.1.3 科室运营总览

**功能描述**：
以科室为单位展示运营数据，支持科室横向对比，发现运营亮点和问题科室。

**核心功能**：

| 功能点 | 说明 |
|--------|------|
| 科室收入排名 | 按收入降序展示各科室收入，支持切换人次/工作量排名 |
| 科室收入结构 | 各科室收入构成分析（药品、材料、检查、手术等） |
| 科室效率对比 | 各科室床位使用率、平均住院日、周转次数对比 |
| 科室质量对比 | 各科室质量指标对比（质控率、合格率等） |
| 下钻分析 | 点击科室名称，下钻到科室详细分析 |
| 语音查询 | "骨科收入排名"、"心内科效率如何"等语音查询 |

#### 3.1.4 DRG运营总览

**功能描述**：
展示全院DRG核心指标和盈亏情况，支持病组维度和科室维度分析。

**核心功能**：

| 功能点 | 说明 |
|--------|------|
| CMI值展示 | 全院CMI值及同比变化 |
| 入组率监控 | 入组率及未入组病例分析 |
| 结余率展示 | 盈利病组占比、亏损病组占比 |
| 病组分布 | 主要病组分布和盈亏情况 |
| DRG盈亏TOP | 盈利TOP5病组、亏损TOP5病组 |
| 时间费用指数 | 时间消耗指数、费用消耗指数展示 |
| 语音查询 | "CMI值多少？"、"哪个病组亏损最多？"等语音查询 |

#### 3.1.5 成本概览

**功能描述**：
展示全院成本数据和结构，支持成本四级分摊结果展示。

**核心功能**：

| 功能点 | 说明 |
|--------|------|
| 成本总量展示 | 本月总成本及同比变化 |
| 成本结构分析 | 人员成本、药品成本、耗材成本、其他成本占比 |
| 成本趋势 | 近12个月成本变化趋势 |
| 科室成本排名 | 各科室成本排名，支持按成本率排序 |
| 分摊结果 | 四级分摊法成本分摊结果展示 |
| 语音查询 | "本月成本多少？"、"哪个科室成本最高？"等语音查询 |

#### 3.1.6 预警中心

**功能描述**：
集中展示全院各类预警信息，支持预警原因分析和处理建议。

**核心功能**：

| 功能点 | 说明 |
|--------|------|
| 预警分类 | 运营预警、质量预警、DRG预警、成本预警 |
| 预警分级 | 红色（紧急）、黄色（重要）、蓝色（提示） |
| 预警原因 | AI分析预警产生的根本原因 |
| 建议措施 | AI给出预警处理建议 |
| 预警处理 | 支持标记已处理、转发给负责人 |
| 预警统计 | 预警数量趋势、预警处理率 |
| 语音播报 | 可设置定时语音播报预警信息 |

#### 3.1.7 大屏管理

**功能描述**：
管理系统中的可视化大屏，支持自定义布局、主题设置、投放管理。

**核心功能**：

| 功能点 | 说明 |
|--------|------|
| 大屏列表 | 查看所有已创建的大屏 |
| 新建大屏 | 拖拽式可视化编辑器，创建自定义大屏 |
| 布局模板 | 提供多种预设布局模板（单屏、双屏、三屏） |
| 主题设置 | 深色主题、浅色主题、品牌主题 |
| 投放管理 | 大屏投放时段、循环播放设置 |
| 权限管理 | 大屏查看权限控制 |
| 语音控制配置 | 配置大屏语音唤醒词和控制指令 |

### 3.2 目录二：运营分析

**定位**：满足院长和运营部门日常运营数据查询和分析需求

**设计理念**：借鉴用友财务分析 + 卫宁运营分析 + 金蝶移动报表

#### 3.2.1 门诊运营分析

**功能描述**：
提供门诊运营数据的详细分析，支持多维度查询、下钻分析和数据导出。

**核心功能**：

| 功能点 | 说明 |
|--------|------|
| 门诊日报 | 每日门诊运营数据，包括人次、收入、人均费用 |
| 门诊统计 | 按科室、医师、疾病分类统计门诊数据 |
| 门诊趋势 | 门诊人次、收入趋势分析 |
| 科室门诊分析 | 各科室门诊运营指标对比 |
| 医师门诊排名 | 门诊工作量排名前10医师 |
| 疾病谱分析 | 门诊患者疾病分布分析 |
| 时段分析 | 门诊就诊时段分布（上午/下午/夜诊） |
| 来源分析 | 门诊患者来源分析（本地/外地/预约/现场） |
| 数据导出 | 支持导出Excel/PDF |
| 语音查询 | "昨天门诊多少人次？"、"骨科门诊收入"等语音查询 |

**查询条件**：

| 条件类型 | 可选值 |
|----------|--------|
| 时间范围 | 今日、昨日、近7天、近30天、本月、上月、自定义 |
| 科室 | 全院、单个或多个科室 |
| 医师 | 全院、单个或多个医师 |
| 患者类型 | 全部、医保、自费、公费 |

**数据字段**：

| 字段名 | 说明 | 数据来源 |
|--------|------|----------|
| 就诊日期 | 日期 | HIS |
| 科室名称 | 科室 | HIS |
| 医师姓名 | 医师 | HIS |
| 门诊人次 | 人次 | HIS |
| 人均费用 | 元 | HIS/财务 |
| 门诊收入 | 元 | 财务 |
| 药占比 | % | 计算 |
| 检查占比 | % | 计算 |
