# 医院运营决策分析可视化大屏需求文档

> **版本**：v1.0
> **日期**：2026-05-25
> **定位**：院长视角的可视化大屏系统
> **核心理念**：一屏看全院、一屏知全局、一屏做决策

---

## 一、大屏系统总体设计

### 1.1 设计理念

**院长视角六大核心关注点：**

| 关注维度 | 院长关心的问题 | 大屏解决方式 |
|----------|----------------|--------------|
| **运营规模** | 今天看了多少病人？做了多少手术？收入多少？ | 核心指标实时展示，5分钟刷新 |
| **运营趋势** | 本月相比上月增长了吗？同比如何？ | 趋势图表叠加对比，一目了然 |
| **资源效率** | 床位够用吗？手术室利用率高吗？医生忙得过来吗？ | 资源利用率仪表盘，紧张科室预警 |
| **财务健康** | 本月盈亏如何？哪些科室赚钱？成本控制得好吗？ | 收支分析图，盈利/亏损科室排名 |
| **DRG绩效** | CMI值多少？哪些病组亏损？医生DRG能力如何？ | DRG核心指标，盈亏分析TOP榜 |
| **问题预警** | 有什么异常需要关注？AI建议怎么处理？ | 智能预警中心，AI归因+建议 |

### 1.2 大屏分类体系

```
医院运营决策分析可视化大屏体系
│
├── 【综合类大屏】3张
│   ├── 01 医院运营总览大屏（首页默认）
│   ├── 02 运营趋势分析大屏
│   └── 03 智能预警中心大屏
│
├── 【业务类大屏】4张
│   ├── 04 财务管控大屏
│   ├── 05 DRG盈亏分析大屏
│   ├── 06 科室绩效对比大屏
│   └── 07 成本管控大屏
│
└── 【专项类大屏】3张
    ├── 08 医疗质量监控大屏
    ├── 09 资源配置优化大屏
    └── 10 医保结算分析大屏
```

### 1.3 大屏通用功能

| 功能 | 说明 |
|------|------|
| **自动刷新** | 5分钟自动刷新数据，支持手动刷新 |
| **时间切换** | 今日/昨日/本周/上周/本月/上月/自定义 |
| **全屏模式** | 支持全屏投放，隐藏边框 |
| **语音唤醒** | "打开运营大屏"、"切换到财务大屏" |
| **语音查询** | "骨科收入多少？"、"床位使用率如何？" |
| **下钻分析** | 点击数据下钻到详细页面 |
| **对比模式** | 支持同期对比、科室对比 |
| **预警高亮** | 异常数据红色高亮显示 |
| **截图分享** | 一键截图，分享到微信/邮箱 |
| **多屏投放** | 支持投放到院长室、会议室、大厅 |
